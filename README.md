
# Fiftyone setup guide

Install fiftyone

```bash
  pip install fiftyone
```
Your folder structure should look like this
```bash
.
├── clip-features-32/
├── keyframes/
├── map-keyframes/
├── media-info/
├── objects/
├── app.py
├── load_dataset.py
├── start.bat
```

## How to run
Run this .bat file
```bash
start.bat
```
